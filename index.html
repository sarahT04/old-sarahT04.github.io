Coding

Some background story on my coding journey.

It was late December. I had a fever and was ultimately bored with studying. A good friend of mine was showing off a program he made from Python. He showed me some resources on where to learn coding online and independently. The first resource I looked was from FreeCodeCamp's collaboration with Mike, a video titled "Learn Python - Full Course for Beginners"

Since Christmas 2020, I have been fiddling around with backend coding. Making small programs, games, and utilities to help me with studies such as Math/Physics stuffs

My main language is Python and I am comfortable in JS. But I've also learned other languages such as Node.js, Dart/Flutter for multi-app development, and C#.

So far on my journey, I have learned introductional level of Cyber Security, Web development with Django from Harvard's CS50, some stuffs in Machine Learning (mostly NLP from Coursera), and some Dynamic Programming and DS/Algorithms.

CLI Scheduler

CLI Scheduler, like the name suggests, is a terminal/CMD implementation to see, add, and delete your schedule all in a single line. This project came up to me when I first started using github. I love the one-line way of doing every command, and so I thought "Well, a CLI to schedule my stuffs won't be so hard, huh". Using JSON and Python, this was written with CRUD (well technically CRD) in mind.

CI/CD with Vigenere

This project was just me trying out TDD or Test Driven Development. I first heard this from a software engineering class. The teacher was showing us its importance and he felt that it's such a shame that TDD not being emphasized a lot during his studies in universities. I got curious, and I tried out TDD with Vigenere Cipher as my playground. The program is a simple Login/Register with Vigenere Cipher as encryption, SQL as data storing, and Python as the backend. There's also an additional username and password checker/requirements ;)
